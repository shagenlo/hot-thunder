★"{{firebase_uid}}":
"type":

object

"title":

{{firebase_uid}} schema.

"description":

The 'firebase_uid' key is a duplicate of the UID assigned when the persons entry in the Firebase authentication system is performed. In this manner, we can perform fast lookups on the /Users node using the persons official Firebase Auth UID. A Firebase UID looks like this: 6O6qCG5TBFZgnefCEjI8tKkNTkG2
